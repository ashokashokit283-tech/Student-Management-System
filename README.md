# Student Management System (Java)

##  Description
The Student Management System is a console-based Java application that helps manage student records.
It allows users to add, view, update, delete, and search student details using a menu-driven approach.

This project is developed as a **mini project** to practice **core Java concepts** such as OOP, collections, and user input handling.



## Features
- Add new student details
- View all students
- Update existing student information
- Delete a student record
- Search student by ID
- Menu-driven console interface



## Technologies Used
- Java
- Eclipse IDE
- ArrayList
- Iterator
- OOP Concepts (Classes, Objects, Methods)
